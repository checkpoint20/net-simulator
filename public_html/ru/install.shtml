Системные требования

Для работы NET-Simulator необходимо, что бы на вашем компьютере была установлена виртуальная машина Java (JVM). JVM обычно установливается вместе с дистрибутивом Linux. Пользователям Windows необходимо скачать и установить JVM отдельно.

Существуют два варианта JVM: Java Runtime Environment (JRE) и Java Development Kit (JDK). Если не знаете какой вариант выбрать, скачивайте JRE.

Вы можете проверить, установлена ли JVM на вашем компьютере следующим образом. Введите команду java -version в коммандной строке (Windows) или терминале (Mac/Unix/Linux). Если JVM установлена, то в результате вы увидете вывод похожий на изображенный на картинке ниже.

Установка

Просто разархивируйте скачанный архив. Откройте папку NET-Simulator и запустите run.bat на Windows или run.sh на Linux.

Известные проблемы

Не устанавливайте NET-Simulator в папку, в имени которой содержатся не ASCII символы. Например, не стоит использовать папку 'C:\программы' или 'C:\Documents and Settings\user\Рабочий стол'. Иначе вы можете получить ошибку при старте программы:
