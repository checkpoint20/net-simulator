System requirements

To run NET-Simulator you need to have a Java Virtual Machine (JVM) version 8+ installed on your system.

On the download page you will find two kinds of JVM: Java Runtime Environment (JRE) and Java Development Kit (JDK). If you are not sure, download and install JRE.

To check if you have Java installed on your system, type: java -version in a command prompt (Windows) or console (Mac/Unix/Linux). If JVM is installed properly you will see an output as shown below.

Installation

Just unzip downloaded archive. Open NET-Simulator folder and run run.bat on Windows or run.sh on Linux.
